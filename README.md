# README
## ●サービス概要
Google Vision APIの「Text Detection」検証用サービスです。

## ●サービスURL
https://textdetectiontest.herokuapp.com/

## ●機能
- Vision APIに画像を送り、レスポンスを確認できる
  - 連続でアップロードすると、表示内容が新しく切り替わる
- 画像をAWS S3に保存できる
